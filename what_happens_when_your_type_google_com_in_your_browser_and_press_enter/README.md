# What Happens When You Type Google.com in Your Browser

## Description

This project explains what happens behind the scenes when a user types **https://www.google.com** into a web browser and presses **Enter**.

The article describes the complete request flow, from DNS resolution to rendering the webpage in the browser. It also includes a diagram illustrating the communication between the browser, servers, and database.

## Topics Covered

- DNS Request
- TCP/IP Connection
- HTTPS / TLS
- Firewall
- Load Balancer
- Web Server
- Application Server
- Database
- Browser Rendering
